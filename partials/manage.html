    Unfortunately, scanning your memory card's dictdata directory for dictionaries was not successful.

    Please make sure you inserted the (right) memory card and place your dictionaries in a directory named dictdata directly at your memory card's root.

    Furthermore, make sure your dictionary is in proper StarDict format, i.e. containing at least one file of each of the following types/file endings: idx, ifo, dict.

    • dictdata/testDict/mydict.dict
    • dictdata/testDict/mydict.idx
    • dictdata/testDict/mydict.ifo

    • dictdata/testdict2/a2ndDict.idx
    • dictdata/testdict2/a2ndDict.ifo

    Note that files may be compressed, i.e. having an additional dz at the end.
